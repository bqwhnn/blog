# 指针

## 有效的指针运算

1. 相同类型指针之间的赋值运算
2. 指针同整数之间的加减运算
3. 指向相同数组中元素的两个指针间的减法或比较运算（相减得到二者之间的元素个数，==、!=、<、>= 等可以针对二者在数组中的位置进行比较）
4. 将指针赋值为 0 或指针和 0 之间的比较运算

不经过强制类型转换而直接把指向一种类型对象的指针赋值给指向另一种类型对象的指针是不合法的，除非其中之一是 void * 类型的指针。
